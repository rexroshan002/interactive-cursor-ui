# Interactive Cursor UI

A minimal and modern interactive cursor experience built using pure HTML, CSS, and JavaScript.

This project features a dynamic circular cursor that reveals masked text using `clip-path`, along with a Light/Dark mode toggle animation.

---

## ✨ Features

- Custom animated cursor
- Text masking using CSS `clip-path`
- Light / Dark mode toggle
- Smooth hover scaling effect
- CSS Variables for dynamic theming
- requestAnimationFrame animation loop

---

## 🛠 Tech Stack

- HTML5
- CSS3 (Custom Properties, clip-path)
- Vanilla JavaScript
- requestAnimationFrame API
